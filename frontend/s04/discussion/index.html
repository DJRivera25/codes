HTML Images

An image can be shown using <img> tag along with its src

<img src ="https://res.cloudinary.com/dneeqo5vq/image/upload/v1708476720/dog300x300_yzrind.jpg">

<img src="./images/ZuittLogo.jpg">

Valid image extension are the following

  1. png
  2. jpg
  3. jpeg
  4. gif
  5. svg
  6. webp
